Project1: Population Segmentation Report

Objective

Aim: The objective of the Population Segmentation Report is to provide a structured, data-driven overview of member utilization and risk patterns across various dimensions, enabling compliance with state reporting requirements and supporting strategic decision-making in healthcare management.

ðŸŽ¯ Main Requirements

1. Regulatory Compliance

  • Fulfill biannual reporting requirements to the state
  • Ensure transparency and accountability in member care and cost patterns

2. Population Health Insights

  • Segment members by line of business (LOB), provider, demographics, and utilization
  • Identify low-risk populations (e.g., those with only preventive visits or no claims)
  • Flag high-cost or high-utilization members for care coordination or case management

3. Provider and Facility Performance

  • Analyze ER and inpatient activity by facility, diagnosis grouping, and PCP attribution
  • Support provider engagement and performance improvement initiatives

4. Cost and Risk Stratification

  • Track members with claims exceeding thresholds (e.g., $70,000)
  • Identify patterns in ER visits and inpatient admits to inform risk mitigation strategies

5. Trend Analysis

  • Enable year-to-date (YTD) comparisons across reporting periods
  • Monitor shifts in member behavior, provider performance, and cost drivers

Source code for MembersWithERVisits dataset

-- 1. Declare Date Variables DECLARE @start_date DATE; DECLARE @end_date_enroll DATE; DECLARE @end_date_dos DATE; DECLARE @start_date_int INT; DECLARE @end_date_int_enroll INT; DECLARE @end_date_int_dos INT; -- 2. Set Dates Based on Current Month IF MONTH(GETDATE()) IN (9, 10, 11, 12) BEGIN -- Use current year SET @start_date = CAST(CAST(YEAR(GETDATE()) AS VARCHAR) + '-01-01' AS DATE); SET @end_date_enroll = CAST(CAST(YEAR(GETDATE()) AS VARCHAR) + '-12-01' AS DATE); SET @end_date_dos = CAST(GETDATE() AS DATE); END ELSE BEGIN -- Use previous year SET @start_date = CAST(CAST(YEAR(GETDATE()) - 1 AS VARCHAR) + '-01-01' AS DATE); SET @end_date_enroll = CAST(CAST(YEAR(GETDATE()) - 1 AS VARCHAR) + '-12-01' AS DATE); SET @end_date_dos = CAST(CAST(YEAR(GETDATE()) - 1 AS VARCHAR) + '-12-31' AS DATE); END -- 3. Convert Dates to Integer Format (YYYYMMDD) SET @start_date_int = CAST(CONVERT(VARCHAR, @start_date, 112) AS INT); SET @end_date_int_enroll = CAST(CONVERT(VARCHAR, @end_date_enroll, 112) AS INT); SET @end_date_int_dos = CAST(CONVERT(VARCHAR, @end_date_dos, 112) AS INT); -- 4. CTE: Members Enrolled in Date Range WITH Members AS ( SELECT f.MemberSK, f.ProgramID, p.[Program Name] FROM dbo.FactEnrollment f WITH (NOLOCK) LEFT JOIN dbo.DimProgram p WITH (NOLOCK) ON f.ProgramID = p.ProgramID WHERE f.Mem_YYYYMM BETWEEN @start_date_int AND @end_date_int_enroll GROUP BY f.MemberSK, f.ProgramID, p.[Program Name] ), -- 5. CTE: Members with 3+ ER Visits MembersWithERVisits AS ( SELECT p.[Program Name] AS Program, a.MemberSK, f.PCPProvSK AS pcp, f.AgeGroupSK AS agegr, f.ClaimID_Base, f.PrimaryDiag, f.DOS_YYYYMM, COUNT(DISTINCT a.StartDateHeader) AS Visits FROM ( SELECT MemberSK, ClaimID_Base, MAX(ERFlag1) AS ERFlag, StartDateHeader, ProgramSK FROM dbo.FactClaim WITH (NOLOCK) WHERE ERFlag1 = 1 AND StartDateHeader BETWEEN @start_date_int AND @end_date_int_dos GROUP BY MemberSK, StartDateHeader, ProgramSK, ClaimID_Base ) a JOIN Members m ON m.MemberSK = a.MemberSK JOIN dbo.DimProgram p ON p.ProgramSK = a.ProgramSK JOIN dbo.FactClaim f ON m.MemberSK = f.MemberSK GROUP BY p.[Program Name], a.MemberSK, f.PCPProvSK, f.AgeGroupSK, f.ClaimID_Base, f.PrimaryDiag, f.DOS_YYYYMM HAVING COUNT(DISTINCT a.StartDateHeader) >= 3 ) -- 6. Final Select: Output Members with 3+ ER Visits SELECT MemberSK, Program, agegr, pcp, ClaimID_Base, PrimaryDiag, DOS_YYYYMM FROM MembersWithERVisits GROUP BY Program, MemberSK, ClaimID_Base, pcp, agegr, PrimaryDiag, DOS_YYYYMM;

Selected filters =
VAR MaxFilters = 3

RETURN
    IF (
        ISFILTERED ( DimDate[Date] ),
        VAR __f1 = FILTERS ( DimDate[Date] )
        VAR __r1 = COUNTROWS ( __f1 )
        VAR __t1 = TOPN ( MaxFilters, __f1, DimDate[Date] )
        VAR __d1 = CONCATENATEX ( __t1, DimDate[Date], ", " )
        VAR __x1 =
            "Date: " & __d1 &
            IF ( __r1 > MaxFilters, ", ... [" & __r1 & " items selected]" ) & " "
        RETURN __x1 & UNICHAR(13) & UNICHAR(10)
    )
    &
    IF (
        ISFILTERED ( DimDate[FullDateAlternateKey] ),
        VAR __f2 = FILTERS ( DimDate[FullDateAlternateKey] )
        VAR __r2 = COUNTROWS ( __f2 )
        VAR __t2 = TOPN ( MaxFilters, __f2, DimDate[FullDateAlternateKey] )
        VAR __d2 = CONCATENATEX ( __t2, DimDate[FullDateAlternateKey], ", " )
        VAR __x2 =
            "Full date: " & __d2 &
            IF ( __r2 > MaxFilters, ", ... [" & __r2 & " items selected]" ) & " "
        RETURN __x2 & UNICHAR(13) & UNICHAR(10)
    )
